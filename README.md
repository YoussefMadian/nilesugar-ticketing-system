# NS Ticketing System

NS Ticketing System is an internal IT Service Management application developed using Microsoft Power Platform to manage IT support operations across Nile Sugar.

The system enables IT engineers to create, track, and resolve support tickets while providing reporting, asset tracking, and SLA monitoring capabilities.

---

## Technologies Used

- Microsoft Power Apps
- Power Automate
- SharePoint Online
- Microsoft Entra ID (Azure AD)
- Office365 Users Connector
- Outlook Connector

---

## Core Features

### Ticket Management
- Create IT support tickets
- Assign engineers
- Track ticket status
- Document root cause and solution
- SLA monitoring
- Issues Reporting
- Manage Assets
- Manage Consumables

### Reporting Dashboard
Interactive reports including:

- Tickets by Status
- Tickets by Area
- Tickets by Site
- Tickets by Type
- Engineer workload
- Top 5 users generating tickets
- KPI Cards
- SLA Compliance
- Recent Tickets
- Tickets per Month
- Tickets per Year

### Asset Tracking
Track movement of IT devices between locations.

### Consumables Tracking
Monitor IT consumables usage across departments.

### User Ticket Search
IT engineers can search tickets by user name or email.

### IT Team Directory
Employees can view the IT team directory including contact information.

---

## System Modules

1. Ticket Management
2. User Ticket Portal
3. IT Dashboard
4. Asset Tracking
5. Consumables Tracking
6. IT Team Directory

---

## Notifications

Automated email notifications are triggered when tickets are created.

Recipients include:

- Area Manager
- User Manager
- Infrastructure Manager
- Service Desk Manager
- Assigned Engineer
- Affected User

---

## SLA Monitoring

The system monitors ticket resolution against defined SLA targets using the following fields:

- SLA Target
- Resolved Time
- SLA Breached
