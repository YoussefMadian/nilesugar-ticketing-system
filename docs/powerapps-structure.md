Screens:

Home Screen
Navigation to all system modules.

Create Ticket Screen
Used by IT engineers to create new support tickets.

My Tickets Screen
Allows users to track their own tickets.

Ticket Details Screen
Displays detailed information about a ticket.

IT Dashboard Screen
Contains system reports and analytics.

IT Team Screen
Displays IT engineers directory.

User Search Screen
Allows IT staff to search tickets by user.

Assets Tracking Screen
Tracks movement of devices between locations.

Consumables Screen
Tracks consumables usage.
